3. Luồng ReAct Agent (Kiến trúc B) — Chi tiết

Điểm mấu chốt #1 — System Prompt (agent.py:21)

Đây là "bộ não" của Agent. Prompt liệt kê tên tool + mô tả để LLM biết khi nào gọi tool nào. LLM không tự nhiên biết tool — nó chỉ in ra text theo đúng format Action: tên(json).

Tinh chỉnh: Thêm ví dụ few-shot vào prompt để LLM ít mắc lỗi format hơn.

Điểm mấu chốt #2 — Regex Parser (agent.py:87)

Pattern: r"Action:\s*(\w+)\((\{.*?\})\)"

Đây là điểm dễ vỡ nhất. Nếu LLM xuất ```json{...}``` thay vì JSON thô → regex không khớp → PARSER_ERROR.

Fix: Thêm bước tiền xử lý strip markdown trước khi regex.

Điểm mấu chốt #3 — max_steps (agent.py:15)

Mặc định max_steps=5. Mỗi bước = 1 lần gọi LLM. Câu hỏi cần 2 tool = ít nhất 3 bước (gọi tool A, gọi tool B, tổng hợp).

Tinh chỉnh: Tăng lên 7-8 cho câu hỏi phức tạp. Giảm xuống 3 để tiết kiệm chi phí.

Điểm mấu chốt #4 — current_context (agent.py:58)

Context được tích lũy qua từng bước. LLM đọc toàn bộ lịch sử Thought→Action→Observation trước đó để quyết định bước tiếp theo.

Nguy hiểm: Nhiều bước → context rất dài → tốn token → chậm.

4. Hệ Thống Tool — Chi tiết từng lớp

4.1 Cách _execute_tool() hoạt động từng dòng

Bước 1 — Tìm tool (agent.py:115)

Duyệt list self.tools, so sánh tool['name'] == tool_name. Tên phải khớp chính xác với key "name" trong AVAILABLE_TOOLS.

Lỗi thường gặp: LLM viết sai tên (viết hoa, dấu gạch dưới). Fix bằng cách thêm fuzzy match.

Bước 2 — Parse JSON (agent.py:119)

json.loads(args) chuyển string "{\"student_id\": \"HS001\"}" thành dict Python.

Lỗi thường gặp: LLM dùng nháy đơn thay nháy đôi → JSONDecodeError. Cần pre-process thay ' → ".

Bước 3 — Validate Pydantic (agent.py:122)

tool["input_model"](**parsed_args) — ép kiểu và kiểm tra field bắt buộc. Nếu LLM quên truyền student_id → ValidationError → trả về error string cho LLM đọc và sửa.

Bước 4 — Gọi hàm thực (agent.py:125)

tool["function"](**validated_data.model_dump()) — gọi hàm Python thực sự (ví dụ get_student_academic_records(student_id="HS001")), hàm truy vấn mock_db và trả về JSON string.
